# Drill_Bit_Defect_Detection
- Tài liệu quá trình thực hiện: https://canva.link/zr4io2buqf924sv
- EDA: lấy được insight được từ dữ liệu để biết trước khi tiền xử lý dữ liệu
- Data Preprocessing: chuyển đổi định dạng yolo, cắt vùng lỗi chính để tối ưu hình khi train -> giúp train nhanh hơn -> ít tốn bộ nhớ

## Kỹ năng học được: 
- debug lỗi bằng việc print, visualize data trước khi chạy data thật để tránh làm hỏng data
- đọc tài liệu nghiên cứu qua những bài báo
- có khả năng giải thích code mình viết/ AI viết để người khác hiểu

### AI hỗ trợ: 
- Antigrivity, Geimini,... nhiệm vụ lên kế hoạch trước khi làm, đọc hiểu source code và debug lỗi
